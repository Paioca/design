# RevTrue — Marca

## Conceito

A RevTrue é uma consultoria de vendas. A marca foi construída sobre a tensão produtiva entre **rev** e **true**:

- **rev** — energia, ação, movimento, agilidade. Cor: laranja avermelhado `#E8693B`.
- **true** — sinceridade, confiança, profissionalismo. Cor: charcoal `#14181F`.

A combinação traduz a promessa da consultoria: agir rápido, mas com base em diagnóstico real.

## Palavras-chave

Crescimento · Energia · Confiança · Agilidade · Transformação

## Narrativa visual

A identidade conta uma história em três tempos:

1. **Analisar o panorama** — para resultados reais é preciso enxergar o todo
2. **Identificar o ponto de melhoria** — onde a operação trava
3. **Agir de forma efetiva** — para gerar resultado

## Logo

A marca tem 3 versões principais e 2 versões de ícone:

| Versão | Quando usar |
|---|---|
| `logo-primary` (laranja + charcoal) | Padrão, sobre fundos claros |
| `logo-charcoal` (monocromática escura) | Fundos claros, peças institucionais sóbrias |
| `logo-orange` (monocromática laranja) | Fundos escuros, peças com energia alta |
| `icon-bicolor` | Avatar, favicon, espaços pequenos |
| `icon-orange` | Sobre fundo escuro / charcoal |

### Detalhes simbólicos

- **Seta na letra "e"** — ponta ascendente diagonal, comunica progresso e trajetória positiva
- **Equilíbrio entre "rev" e "true"** — pesos diferentes que somam harmonia (light + bold)
- **Kerning amplo** — leitura fluida, reforça a ideia de "descomplicado"

## Tipografia

- **Hongkong** (display) — bastonada, geométrica, sem serifas. Modernidade + acessibilidade.
- **Roboto / Roboto Condensed** (UI) — neutra, legível, escalável.

## Tom de voz

- Direto e prático ("descomplicado")
- Confiante, sem ser arrogante
- Foca em ação e resultado mensurável
- Evita jargão corporativo vazio

## Do / Don't

| ✅ Faça | ❌ Evite |
|---|---|
| Usar laranja em ações decisivas (CTAs, destaques) | Usar laranja em blocos longos de texto |
| Combinar Hongkong (títulos) + Roboto (corpo) | Misturar mais de duas famílias tipográficas |
| Manter respiro generoso ao redor do logo | Apertar o logo perto de outros elementos |
| Usar gradiente `rev-true` em CTAs heroicos | Usar o gradiente em cada seção (perde força) |
| Aplicar logo monocromático sobre fundos com pouco contraste | Distorcer, rotacionar ou recolorir o logo fora das versões oficiais |
